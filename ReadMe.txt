如果仅仅想要浏览网页，可以直接浏览器输入IP地址：http://47.101.205.176:5000/

使用方法：
1.python 运行 data_parser.py
2.打开浏览器localhost:5000

可能出现的问题：

1.运行python时编码方式错误，不能够正确解析 
解决办法：把static/data/data_preprocessed 以另一种格式（ANSI->UTF-8或是UTF-8->ANSI）另存为当前目录下。

2.视图有一部分被遮挡 
解决办法：浏览器切换到Chrome，并F11 全屏浏览
